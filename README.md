## Features

- **Workspace Scanning:** Automatically detects `SKILL.md` files in typical AI assistant directories (`.agent/skills`, `.cursor/rules`, custom paths).
- **Gap Analysis:** Visually compare the skills active in your current project against your available library.
- **One-Click Imports:** Missing a critical skill? Import it directly from the UI or via hovering over the skill name.
- **Native VS Code UI:** Integrated using standard VS Code Sidebars, Tree Views, QuickPicks, and Status Bars.
- **Works with Forks:** Fully compatible with Cursor, VSCode, and Antigravity and etc.
